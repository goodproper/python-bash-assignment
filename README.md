# Python Assignment

This repository contains four simple Python programs:

## 1. Grade Checker
Takes a score (0-100) and prints grade:
A, B, C, D, or F.

Run with:
python grade checker.py

## 2. Student Grades
Uses a dictionary to store student names and grades.
Options to add, update, and print.

Run with:
python studentgrades.py

## 3. Write to a File
Asks for filename and text, then writes it to the file.

Run with:
python write.py

## 4. Read from a File
Reads and displays the content of a file.

Run with:
python readfile.py
